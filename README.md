# PhishSim-Security-Awareness-Platform
PhishSim is an open-source phishing email simulator designed for 
security awareness training. It allows security teams to create 
realistic phishing campaigns, track who opens and clicks emails, 
and measure organizational susceptibility to social engineering attacks.

Features:
- 4 prebuilt phishing email templates (IT, CEO fraud, HR, DocuSign)
- Real-time open and click tracking via token-based pixels
- Campaign management dashboard with live event feed
- SMTP support for sending real simulation emails
- SQLite database — no setup required
- Awareness landing page shown to users who click phishing links
- Risk reporting per campaign

Built with: Python, Flask, SQLite, HTML/CSS/JavaScript
